# Fundamentos de SEO

> Princípios para visibilidade em motores de busca.

---

## 1. Framework E-E-A-T

| Princípio          | Sinais                                          |
| ------------------ | ----------------------------------------------- |
| **Experiência**    | Conhecimento de primeira mão, exemplos reais    |
| **Expertise**      | Credenciais, profundidade de conhecimento       |
| **Autoridade**     | Backlinks, menções, reconhecimento da indústria |
| **Confiabilidade** | HTTPS, transparência, informação precisa        |

---

## 2. Core Web Vitals

| Métrica | Meta    | Mede                        |
| ------- | ------- | --------------------------- |
| **LCP** | < 2.5s  | Performance de carregamento |
| **INP** | < 200ms | Interatividade              |
| **CLS** | < 0.1   | Estabilidade visual         |

---

## 3. Princípios de SEO Técnico

### Estrutura do Site

| Elemento       | Propósito               |
| -------------- | ----------------------- |
| XML sitemap    | Facilita o rastreamento |
| robots.txt     | Controla o acesso       |
| Canonical tags | Previne duplicatas      |
| HTTPS          | Sinal de segurança      |

### Performance

| Fator                | Impacto          |
| -------------------- | ---------------- |
| Velocidade da página | Core Web Vital   |
| Mobile-friendly      | Fator de ranking |
| URLs limpas          | Rastreabilidade  |

---

## 4. Princípios de SEO de Conteúdo

### Elementos da Página

| Elemento         | Melhor Prática                            |
| ---------------- | ----------------------------------------- |
| Title tag        | 50-60 caracteres, palavra-chave no início |
| Meta description | 150-160 caracteres, compelling            |
| H1               | Um por página, palavra-chave principal    |
| H2-H6            | Hierarquia lógica                         |
| Alt text         | Descritivo, sem keyword stuffing          |

### Qualidade de Conteúdo

| Fator        | Importância            |
| ------------ | ---------------------- |
| Profundidade | Cobertura abrangente   |
| Atualização  | Atualizações regulares |
| Unicidade    | Valor original         |
| Legibilidade | Escrita clara          |

---

## 6. Diretrizes de Conteúdo com IA

### O que o Google procura

| ✅ Faça                     | ❌ Não faça                  |
| --------------------------- | ---------------------------- |
| IA rascunha + edição humana | Publica conteúdo bruto da IA |
| Adicione insights originais | Copia sem agregar valor      |
| Revisão por especialista    | Ignora verificação de fatos  |
| Siga E-E-A-T                | Keyword stuffing             |

---

## 7. Fatores de Ranking (Priorizados)

| Prioridade | Fator                                   |
| ---------- | --------------------------------------- |
| 1          | Conteúdo de qualidade e relevante       |
| 2          | Backlinks de sites autoridades          |
| 3          | Experiência da página (Core Web Vitals) |
| 4          | Otimização mobile                       |
| 5          | Fundamentos de SEO técnico              |
